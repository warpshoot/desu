# pxlcam ユーザーガイド (User Manual)

カメラ映像や画像をリアルタイムでピクセルアート（ドット絵）に変換するWebツール。

## 📸 モード選択
起動時に「CAM（カメラ）」か「IMG（画像）」を選択。
*   **CAM**: デバイスのカメラ映像をリアルタイム変換。
*   **IMG**: デバイス内の画像を読み込んで変換。
    *   **ズーム**: PCはマウスホイール、スマホはピンチイン/アウト。
    *   **パン (移動)**: ドラッグ or スワイプで切り抜き位置を調整可能。

## 🎨 PALETTE
使用するカラーパレットを選択。レトロゲーム風（4ビット/8ビット）や、ブラウン管、ドットプリンタのような各種カラー制限に変更可能。

## 📏 GRID
ピクセルアートの解像度（細かさ）を選択（8〜128）。数値が小さいほど粗いドット絵になる。

## 🎚️ 色彩調整
各種スライダーで色味を調整。
*   **BRIGHT**: 明るさ（-100〜100）。
*   **CONTST**: コントラスト（0.5〜3.0）。
*   **SATUR**: 彩度（0〜3.0）。
*   **HUE**: 色相（-180〜180）。

## ⚙️ 各種スイッチ
画面下部のトグルやボタン。
*   **DITHER**: ディザリング（網点表現）のON/OFF。限られた色数でグラデーションを滑らかに見せる。
*   **GRID**: キャンバス上のグリッド線の表示ON/OFF。
*   **⟲（カメラ切替）**: フロント/リアカメラの切り替え（CAMモード時のみ）。

## ⏸️ アクション
*   **FREEZE / RESUME**: カメラ映像の一時停止と再開。
*   **LOAD IMG**: 別の画像を新しく読み込む（押すとIMGモードに切り替わる）。

## 💾 Save
変換結果をPNG画像として保存。
プルダウンから保存時の解像度（320px / 640px / 1280px）を選択すると、そのまま保存が実行される。
