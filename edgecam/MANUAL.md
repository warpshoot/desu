# DESU™ edgecam ユーザーガイド

カメラ映像や画像からエッジ（輪郭）を抽出し、さまざまなエフェクトを重ねて出力するWebツール。

---

## 起動

- **CAM** — カメラを起動して映像をリアルタイム処理
- **IMG** — 端末から画像を読み込んで処理

---

## カメラ操作

| ボタン | 機能 |
|--------|------|
| FREEZE | 現在フレームを静止。パラメータ調整に便利 |
| RESUME | 静止を解除してライブ映像に戻る |
| ⇄（カメラ切替） | フロント / リアカメラを切り替え |
| LOAD IMG | 別の画像を読み込む（IMGモード） |

---

## パラメータ（スライダー）

| スライダー | 範囲 | 説明 |
|-----------|------|------|
| LEVELS | 2〜24 | 輪郭の段数。低いほどシンプル、高いほど細かい |
| RES | 64〜512 | 処理解像度。低いほど荒くドット感が出る |
| SMOOTH | 0〜8 | ブラー量。高いほど輪郭がなめらかになる |

---

## 描画モード

2つのモードはどちらか一方のみ選択可能。

| トグル | 説明 |
|--------|------|
| SOLID | 面で塗りつぶした輪郭を描画 |
| BAND | 帯状のストライプで輪郭を描画 |

---

## エフェクト

複数同時に有効化できる（一部は排他）。

| トグル | 説明 | 排他 |
|--------|------|------|
| GHOST | 過去フレームを残像として重ねる | GLITCH |
| MIRROR | 出力を左右反転する | — |
| GLITCH | デジタルグリッチノイズを加える | GHOST |
| PULSE | LEVELS値がランダムに脈動する | — |
| WARP | 多重サイン波で画像を歪める | — |
| SORT | ソース画像をエッジ検出前にピクセルソートする | — |
| FEEDBACK | 前フレームの出力を次フレームの入力に混ぜる | — |

---

## カラー

輪郭の描画色をスウォッチで選択。

WHITE / GREEN / AMBER / CYAN / BLUE / RED / PINK / **MULTI**（多色モード）

---

## 画像操作（IMGモード）

| 操作 | 動作 |
|------|------|
| ドラッグ | パン（移動） |
| ピンチ / マウスホイール | ズーム（0.25x〜8x） |
| ダブルタップ | ズームリセット・センタリング |

---

## 保存・書き出し

右上の保存ボタンから書き出しモーダルを開く。

| 設定 | 選択肢 |
|------|--------|
| サイズ | 320 / 640 / 1280 px |
| 背景 | BLACK（黒） / CLEAR（透過） |

| ボタン | 動作 |
|--------|------|
| PNG | PNGファイルをダウンロード |
| SVG | SVGベクターファイルをダウンロード |
| COPY | クリップボードにコピー |

---

## UI サイズ

左上のステータスバー横の `x1` ボタンで **x1 / x2** を切り替え。
小さい画面でも操作しやすくなる。
