# DESU™ pattern ユーザーガイド (User Manual)

シンプルなWebベースのパターン（幾何学模様）作成ツール。

## 🔷 描画ツール
左下のツールアイコンをタップして選択。
*   **矩形** (3): 長方形・正方形を描画。
*   **円弧** (2): 弧や円を描画。
*   **三角形** (1): 三角形を描画。

キャンバス上でドラッグして図形を配置します。

## 🎨 色
*   **塗りつぶし色** (X): 図形の色を変更。
*   **背景色**: キャンバスの背景色を変更。

## 🔄 対称描画 (S)
対称モードをONにすると、描画が自動的に反転・回転コピーされて幾何学パターンが生成されます。

## #️⃣ グリッド (G)
グリッド線の表示/非表示を切替。

## 💾 保存
右上の ⬇ アイコンをタップ。
*   **全体を保存 (PNG)**: キャンバス全体をPNG保存。
*   **範囲選択**: ドラッグで範囲指定 → 確定 → 保存 or コピー。
*   **倍率**: 1x / 2x / 3x。
*   **背景透過**: チェックで透過PNG出力。

## 📁 ファイル管理
右上のファイルアイコンをタップ。
*   **NEW**: 新規プロジェクト。
*   **SAVE (.json)**: プロジェクトを保存。
*   **LOAD**: JSONファイルから復元。

## 🔍 ズーム & パン
ピンチ操作でズーム。2本指でパン。右下の `⊕` ボタンでリセット。

## ⌨️ ショートカット
*   `1`, `2`, `3`: ツール切替（三角形 / 円弧 / 矩形）。
*   `S`: 対称描画ON/OFF。
*   `G`: グリッドON/OFF。
*   `X`: 塗りつぶし色変更。
*   `⌘S` / `Ctrl+S`: 保存UIを開く。
*   `Del`: 全消去。
*   `⌘Z` / `Ctrl+Z`: 元に戻す。
