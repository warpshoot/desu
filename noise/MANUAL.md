# DESU™ noise ユーザーガイド (User Manual)

画像にノイズ（粒子）エフェクトを加えたアニメーションGIFを生成するWebツール。

## 📷 画像アップロード
画面中央のアップロードエリアをタップして画像を選択、またはドラッグ＆ドロップ。

## 🎚️ Levels
3つのハンドルで明暗の分布を調整。
*   **⚫ 黒点**: 暗い領域の基準。左に寄せると暗部が増加。
*   **🔘 中間**: 全体の明るさバランス。
*   **⚪ 白点**: 明るい領域の基準。右に寄せると明部が増加。

## 🔲 Noise Pattern
1〜5のパターンから選択。数値が大きいほどノイズの粒度や密度が変化します。

## 📐 Scale
出力サイズの倍率を選択。
*   **1x**: 原寸。
*   **2x**: 2倍。
*   **3x**: 3倍。

## 🎬 Generate GIF
設定に基づいてアニメーションGIF を生成。処理中はプログレス表示されます。

## 🔄 Reset
すべての設定を初期状態に戻し、新しい画像を選択可能にする。
